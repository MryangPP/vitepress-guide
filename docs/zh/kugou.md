

  ## 1.开始

 在使用酷狗音乐HiFi前，您需要准备一个有效的酷狗音乐账号。

首次在设备端或 Eversolo Control APP 的“音乐服务”- “流媒体”菜单中打开酷狗音乐时，系统会提示进行账号登录，请使用手机号码完成验证登录。

  ## 2.登录

使用您绑定账号的手机号码，通过短信验证码即可完成登录。

如果您尚未注册酷狗音乐账号，首次使用手机号码登录时，系统会自动为您创建账号。

## 3.浏览

登录成功后，您可以在设备端或 Eversolo Control APP 浏览酷狗音乐。在界面顶部，您将看到 推荐、分类曲库、甄选专辑、我的 等分栏选项。其中 甄选专辑 分栏汇集了高分辨率音源，便于快速查找高品质音乐内容。

您可以逐层浏览这些分类，访问酷狗的各类推荐内容，同时也能在我的分栏选项中查看与设备同步的个人数据，包括您在手机端收藏的歌曲、歌单、歌手、专辑以及最近播放记录。

## 4.搜索

点击顶部搜索框即可在整个酷狗音乐曲库中检索内容，支持按 综合、单曲、歌单、专辑、歌手 等分类搜索，帮助您快速定位心仪的音乐资源。

## 5.更多

在浏览界面中，您可以通过点击曲目右侧的“<img class="imgIcon" src="/icons/kugou_3.png" alt="" />”更多图标，或专辑/歌单右上角的“<img class="imgIcon" src="/icons/kugou_1.png" alt="" />”更多图标，进入更多操作菜单。

该菜单支持以下功能：

| 单曲 更多菜单 | 专辑/歌单 更多菜单 |
| :--- | :--- |
| 添加到设备播放列表 | 现在播放所有曲目 |
| 添加到酷狗音乐HiFi自建歌单 | 所有曲目添加到下一首播放 |
| 查看歌手 | 所有曲目添加到播放队列尾部 |
| 查看专辑 | 添加到设备播放队列 |


**提示：**
+ 收藏的内容可在 “我的” 分栏中随时查看与管理。
+ 添加到设备播放列表的内容，可在我的音乐-设备播放列表中统一查看和管理。

## 6.音质

在酷狗音乐的“我的”分栏中，点击账号信息右上角的设置图标即可设置默认音质。同时，在播放音乐时，也可进入播放详情界面，点击“<img class="imgIcon" src="/icons/kugou_2.png" alt="" />”音质图标即可选择音质模式。

支持的选项包括：
+ 192kHz超母带
+ 96kHz母带
+ Hi-Res
+ 无损（FLAC）
+ 杜比
+ 蝰蛇全景声
