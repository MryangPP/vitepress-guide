

  ## 1.开始

在使用索尼精选 Hi-Res 音乐服务前，您需要准备一个有效的索尼精选账号。
首次在设备端或 Eversolo Control APP的“音乐服务”- “流媒体”菜单中打开索尼精选音乐时，需进入“我的”分栏完成账号登录。

  ## 2.登录
使用您绑定账号的手机号码，通过短信验证码即可完成登录。
如果您尚未注册索尼精选账号，首次使用手机号码登录时，系统会自动为您创建账号。

## 3.浏览

登录成功后，您可以在设备端或 Eversolo Control APP 浏览索尼精选音乐。在界面顶部，您将看到 首页、分类、歌单、排行、我的 等分栏选项。

您可以逐层浏览这些分类，访问索尼精选音乐的各类推荐内容，同时也能在我的分栏选项中查看收藏的单曲、专辑、歌单，以及您自建的各类歌单。

## 4.搜索

点击顶部的搜索框，即可在整个索尼精选音乐曲库中进行检索，查找您想要的 专辑、单曲、歌单、艺术家 等内容，快速定位心仪的音乐资源。

## 5.更多

在浏览界面中，您可以通过点击曲目右侧的“<img class="imgIcon" src="/icons/qqMusic_1.png" alt="" />”更多图标，或专辑/歌单右上角的“<img class="imgIcon" src="/icons/qqMusic_2.png" alt="" />”更多图标，进入更多操作菜单。



该菜单支持以下功能：

| 单曲 更多菜单 | 专辑/歌单 更多菜单(设备端) |
| :--- | :--- |
| 现在播放 / 下一首播放 | 现在播放所有曲目 |
| 添加到播放队列尾部 | 所有曲目添加到下一首播放 |
| 添加到设备播放列表 | 所有曲目添加到播放队列尾部 |
| 收藏到索尼精选音乐 | 添加到设备播放列表 |
| 添加到索尼精选音乐播放列表(自建歌单) |  / |
| 查看艺术家 |  / |


**提示：**
+ 收藏的内容可在 “我的” 分栏中随时查看与管理。
+ 添加到设备播放列表的内容，可在我的音乐-设备播放列表中统一查看和管理。

## 6.音质

在播放索尼精选音乐时，设备会根据当前音源的规格进行直通输出，音质完全取决于此平台原始文件。


支持的选项包括：
+ 192kHz音质
+ 96kHz音质
+ 48kHz音质
