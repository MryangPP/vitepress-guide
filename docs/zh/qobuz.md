

  ## 1.开始

在使用Qobuz音乐服务前，您需要准备一个有效的Qobuz账号。

首次在设备端或 Eversolo Control APP的“音乐服务” - “流媒体”菜单中打开 Qobuz，并按照提示完成账号登录。

  ## 2.登录

请在设备端或 Eversolo Control APP输入您的 Qobuz 账号与密码完成登录。

如您尚未创建 Qobuz 帐号，请点击“Register here”按提示完成账号注册。

## 3.浏览

登录成功后，您可以在设备端或 Eversolo Control APP 浏览Qobuz音乐。在界面顶部，您将看到 Discover、My Weekly Q、My playlists、Genres、Favourites等分栏选项。

您可以逐层浏览这些分类，访问 Qobuz 的各类推荐内容，同时也能在Favourites分栏选项中查看收藏的自建播放列表、最近发行以及艺术家。

点击顶部的搜索框，即可在整个Qobuz曲库中进行检索，查找您想要的 艺术家、专辑、单曲、播放列表 等内容，快速定位心仪的音乐资源。

## 5.更多

在浏览界面中，您可以通过点击曲目右侧的“<img class="imgIcon" src="/icons/qqMusic_1.png" alt="" />”图标，或专辑/歌单右上角的“<img class="imgIcon" src="/icons/qqMusic_2.png" alt="" />”图标，进入更多操作菜单。




该菜单支持以下功能：

| 单曲 更多菜单 | 专辑/歌单 更多菜单 |
| :--- | :--- |
| 现在播放 / 下一首播放 | 现在播放所有曲目 |
| 添加到播放队列尾部 | 所有曲目添加到下一首播放 |
| 添加到设备播放列表 | 所有曲目添加到播放队列尾部 |
| 收藏到Qobuz | 添加到设备播放列表 |
| 添加到 Qobuz 播放列表 | 收藏到 Qobuz |
| 查看艺术家 /曲目信息 | 所有曲目添加到 Qobuz播放列表 |


**提示：**
+  收藏的内容可在 “My Collection” 分栏中随时查看与管理。
+  添加到设备播放列表的内容，可在我的音乐-设备播放列表中统一查看和管理。

## 6.音质
在使用 Qobuz 音乐服务时，您可以在首页点击右上角“<img class="imgIcon" src="/icons/tidal_1.png" alt="" />”图标，或在播放详情界面点击“<img class="imgIcon" src="/icons/qqMusic_3.png" alt="" />”图标来选择音质模式。



支持的选项包括：
+ Low
+ Medium（128 kb/s）
+ Hight（320 kb/s）
+ Lossless（CD-Quality）
+ HiRes（MAX/Master）
