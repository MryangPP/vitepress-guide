

  ## 1.开始

在使用KKBOX音乐服务前，您需要准备一个有效的KKBOX账号。

首次请在设备端的“音乐服务”- “流媒体”菜单中打开 KKBOX，并按照提示完成账号登录。

  ## 2.登录

请在设备端输入您的 KKBOX 账号与密码完成登录。

如您尚未创建 KKBOX 帐号，可滑动至页面底部，点击“注册”按提示完成账号注册。




<img src="/images/kkbox_03.png" />

## 3.浏览

登录成功后，您可以在设备端或 Eversolo Control APP 浏览KKBOX音乐。在界面顶部，您将看到 发现、排行榜、曲风情景、电台、我的 等分栏选项。

您可以逐层浏览这些分类，访问KKBOX音乐的各类推荐内容，同时也能在我的分栏选项中查看收藏的单曲、专辑、歌单，以及您自建的各类歌单。

## 4.搜索

点击顶部的搜索框，即可在整个KKBOX曲库中进行检索，查找您想要的单曲、专辑、歌单、艺人等内容，快速定位心仪的音乐资源。

## 5.更多

在浏览界面中，您可以通过点击曲目右侧的“<img class="imgIcon" src="/icons/qqMusic_1.png" alt="" />”更多图标，或专辑/歌单右上角的“<img class="imgIcon" src="/icons/qqMusic_2.png" alt="" />”更多图标，进入更多操作菜单。

该菜单支持以下功能：

| 单曲 更多菜单 | 专辑/歌单 更多菜单 |
| :--- | :--- |
| 现在播放 / 下一首播放 | 现在播放所有曲目 |
| 添加到播放队列尾部 | 所有曲目添加到下一首播放 |
| 添加到设备播放列表 | 所有曲目添加到播放队列尾部 |
| 收藏 KKBOX | 添加到设备播放列表 |
| 加入 KKBOX 播放列表（自建歌单） | / |
| 查看专辑/艺人 | / |


**提示：**
+ 收藏的内容可在 “我的” 分栏中随时查看与管理。
+ 添加到设备播放列表的内容，可在我的音乐-设备播放列表中统一查看和管理。

## 6.音质

在播放KKBOX音乐时，进入播放详情界面，点击“”音质图标即可选择音质模式。支持的选项包括：
+ 320k
+ 16bit CD 音质
+ 24bit Hi-Res 音质

> 注意：并非所有歌曲都支持 24bit Hi-Res，实际可用音质取决于音乐的原始文件品质。
