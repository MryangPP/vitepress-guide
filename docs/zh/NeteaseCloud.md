

  ## 1.开始

在开始使用网易云音乐前，您需要具备以下条件：一个有效的网易云音乐账号；手机已安装并登录网易云音乐App。

首次在设备端的“音乐服务”- “流媒体”菜单中打开网易云音乐时，系统会自动弹出二维码登录界面，等待您使用手机网易云音乐扫码完成登录。



<img src="/images/NeteaseCloud_1.png" />

  ## 2.登录

请打开手机端网易云音乐，使用扫码功能扫描设备端二维码完成绑定与登录。

如果您尚未注册网易云音乐账号，可以先在手机端的网易云音乐 App 中完成注册。




<img src="/images/NeteaseCloud_2.png" />

## 3.浏览

登录成功后，您可以在设备端或 Eversolo Control APP 浏览网易云音乐。在界面顶部，您将看到 发现、歌单、排行榜、HiFi专区、播客、我的 等分栏选项。其中 HiFi 专区 汇集了高分辨率音源，便于快速查找高品质音乐内容。

您可以逐层浏览这些分类，访问网易云音乐的各类推荐内容，同时也能在我的分栏选项中查看与设备同步的个人数据，包括您在手机端创建的歌单、标记为喜欢的歌曲，以及您收藏的各类歌单。

## 4.搜索

点击顶部的搜索框，即可在整个网易云音乐曲库中进行检索，查找您想要的 单曲、歌单、专辑、歌手 等各类内容，快速定位心仪的音乐资源。

## 5.更多

在浏览界面中，您可以通过点击曲目右侧的“<img class="imgIcon" src="/icons/qqMusic_1.png" alt="" />”更多图标，或专辑/歌单右上角的“<img class="imgIcon" src="/icons/qqMusic_2.png" alt="" />”更多图标，进入更多操作菜单。


该菜单支持以下功能：

| 单曲 更多菜单 | 专辑/歌单 更多菜单 |
| :--- | :--- |
| 现在播放 / 下一首播放 | 现在播放所有曲目 |
| 添加到播放队列尾部 | 所有曲目添加到下一首播放 |
| 添加到设备播放列表 | 所有曲目添加到播放队列尾部 |
| 收藏到网易云音乐或歌单 | 添加到设备播放列表 |
| 查看歌手与专辑信息 |  / |


**提示：**
+ 收藏的内容可在 “我的” 分栏中随时查看与管理。
+ 添加到设备播放列表的内容，可在我的音乐-设备播放列表中统一查看和管理。

## 6.音质

在播放网易云音乐时，进入播放详情界面，点击“<img class="imgIcon" src="/icons/qqMusic_3.png" alt="" />”音质图标即可选择音质模式。

支持的选项包括：
+ 超清母带（Master）
+ 高清臻音（Spatial Audio）
+ 无损（SQ）
+ 极高（HQ）
+ 标准
