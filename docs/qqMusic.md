
  ## 1.开始

  在使用QQ音乐服务前，您需要准备一个有效的QQ音乐账号。
首次使用请在设备端的“音乐服务”- “流媒体”菜单中打开QQ音乐，系统会自动弹出二维码登录界面，请根据提示完成登录。

<img src="/images/qqMusic_1.png" />

  ## 2.登录

请根据您的 QQ 音乐账号绑定方式，使用 QQ、微信或 QQ 音乐的扫码功能扫描设备端二维码完成登录。 
如果您尚未注册 QQ 音乐账号，可前往 QQ 音乐官网：https://y.qq.com/ 完成注册。


<img src="/images/qqMusic_2.png" />

## 3.浏览

登录成功后，您即可在设备端或 Eversolo Control APP 浏览 QQ 音乐。界面顶部提供 推荐、专区、歌单、歌手、听书、我的 等分栏选项，其中“专区”又汇集了 QQ音乐 多个 HiFi 板块的高分辨率音源，便于您快速查找高品质音乐内容。

您可以逐层浏览这些分类，访问 QQ 音乐的各类推荐内容；同时，也可以在“我的”分栏中查看与设备同步的个人数据，包括在 QQ 音乐其他平台关注的歌手、已购内容、收藏单曲、最近播放记录、各类收藏及自建歌单等。

## 4.搜索

点击顶部的搜索框，即可在整个网易云音乐曲库中进行检索，查找您想要的 单曲、歌单、专辑、歌手、节目、歌词 等各类内容，快速定位心仪的音乐资源。

## 5.更多

在浏览界面中，您可以通过点击曲目右侧的“”更多图标，或专辑/歌单右上角的“”更多图标，进入更多操作菜单。

该菜单支持以下功能：

| 单曲 更多菜单 | 专辑/歌单 更多菜单 |
| :--- | :--- |
| 现在播放 / 下一首播放 | 现在播放所有曲目 |
| 添加到播放队列尾部 | 所有曲目添加到下一首播放 |
| 添加到设备播放列表 | 所有曲目添加到播放队列尾部 |
| 收藏 | 添加到设备播放列表 |
| 添加到QQ音乐播放列表 | / |
| 查看歌手信息 | |

**提示：**
+ 收藏的内容可在 “我的” 分栏中随时查看与管理。
+ 添加到设备播放列表的内容，可在我的音乐-设备播放列表中统一查看和管理。

## 6.音质

在播放QQ音乐时，进入播放详情界面，点击“”音质图标即可选择音质模式。支持的选项包括：
+ 臻品母带3.0
+ 臻品母带2.0
+ 黑胶音质（仅黑胶专区）
+ SQ无损品质
+ HQ高品质
+ 标准品质
